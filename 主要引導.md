# AI Agent Space

本資料夾是 AI Agent 的長期工作空間。
根目錄：`C:\Users\s7514\source\repos\AIagentSpace`

## 文件分層

| 文件 | 定位 | 何時讀 |
|---|---|---|
| `主要引導.md` | 總入口、地圖（本檔） | 每次任務開始 |
| `AGENT_BOOTSTRAP.md` | 短版必讀入口 | 每次任務開始 |
| `記憶系統規格.md` | 記憶規格 | 修改記憶系統時 |
| `DEVELOPMENT_PRINCIPLES.md` | 開發原則 | 涉及程式碼/架構時 |
| repo `AGENTS.md` | 專案入口 | 進入 repo 後 |

衝突：使用者要求 > repo AGENTS.md 專案事實 > 記憶系統規格 > 開發原則。

## 啟動流程

每次任務開始：

0. 讀 `主要引導.md`（本檔），取得 AI Agent Space 總覽
1. 讀 `AGENT_BOOTSTRAP.md`
2. 讀 repo `AGENTS.md`
3. 讀 `Memories/index/startup-index.json`
4. 讀 `Memories/logs/index.md`（前 10 筆，至多 3000 tokens）
5. 依任務關鍵字搜尋記憶索引（JSONL）
6. 必要時讀 `DEVELOPMENT_PRINCIPLES.md` + 專案適配文件
7. 輸出「原則讀取回執」

**前置總 token 硬上限：5000**（主要引導 + bootstrap + AGENTS.md + startup-index + logs index）
→ 超出時優先壓縮 logs index 與 startup-index，不刪核心規則。

## 原則讀取回執

| 情境 | 要求 |
|------|------|
| 新對話/高風險/架構/DB/安全 | 完整回執：已讀文件、搜尋關鍵字、3~5 條原則、最小範圍、驗證方式 |
| 小修改 | 極簡回執：已讀文件、範圍、驗證 |
| 純問答、不改檔 | 一句話說明已讀 |
