# EventsAdmin獎項欄位綁定調整

## 任務範圍

- 依使用者要求調整 `/EventsAdmin/Content` 的獎項設定欄位綁定。
- 範圍包含共用 `Views/Shared/vue-component/EventContent.cshtml` 與相關 EventAward 設定讀取。

## 主要變更

- `啟用手動指定(條件篩選)` 改綁 `Events.EnableManualAssignment`。
- `抽獎活動` 改綁 `Events.EnableLottery`。
- `啟用隨機抽獎` / `啟用手動抽獎` radio 改用 `Events.ManualLottery`，`false` 為隨機、`true` 為手動。
- `啟用條件篩選` 改綁 `Events.EnableConditionalFilter`。
- `EventAwardController.GetAwardSettings` 補回 `EnableManualAssignment`、`EnableLottery`、`EnableConditionalFilter` 欄位。
- `EventAward/Content.cshtml` 的欄位可用性同步改用新活動設定欄位。

## 驗證結果

- 執行 `dotnet build SmartExpoIoT/SmartExpoIoT.csproj` 成功。
- 建置結果：0 errors；warnings 為既有套件相容性、nullable、obsolete、未使用變數等警告。

## 記憶重點

- `EventContent.cshtml` 是 EventsAdmin/Events 共用 Vue component；獎項設定欄位綁定需改共用元件，不只改 `EventsAdmin/Content.cshtml`。
